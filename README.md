### Prerequisites
* Node >= 10

### How to run
1. Clone the project.

2. Open a terminal and move your directory to the project.

3. Download all dependencies:
```
npm install
```

4. Run the app:
```
npm run start
```

5. Open the app by going to localhost:8000.

### Built With
* [RevealJS](https://revealjs.com/) - The framework that has been used
